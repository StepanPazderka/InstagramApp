# InstagramApp

This is custom app written in Swift and UIKit to imitate Instagram app. It supports:


<ul>
<li>posting new Images saved via FileManager and CoreData</li>
<li>Liking</li>
<li>Bookmarking</li>
<li>Commenting</li>
<li>Changing profile picture</li>
</ul>
